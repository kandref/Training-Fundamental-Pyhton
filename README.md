# Training Fundamental Python

A data analysis project focusing on credit card customer segmentation using Python and clustering techniques.

## Project

### Credit Card Customer Segmentation

Customer segmentation analysis to identify distinct customer groups based on demographic and spending behavior, enabling targeted marketing strategies.

**Dataset:** `Bank_Personal_Loan_Modelling.csv` — 200 customer records

**Features:**
| Feature | Description | Range |
|---------|-------------|-------|
| CustomerID | Unique identifier | 1–200 |
| Gender | Male / Female | — |
| Age | Customer age | 18–70 |
| Annual Income | Income in $k | $15k–$137k |
| Spending Score | Spending behavior score | 1–100 |

**Analysis Steps:**
1. Data loading & quality check
2. Exploratory data analysis & descriptive statistics
3. Distribution analysis (income & age histograms)
4. Customer segmentation using clustering

**Visualizations:**
- Annual income distribution with KDE
- Age distribution histogram
- Customer segment clusters

## Tech Stack

- **Python** — Programming language
- **Pandas** — Data manipulation
- **NumPy** — Numerical computing
- **Matplotlib & Seaborn** — Data visualization

## How to Use

1. Clone this repository
   ```bash
   git clone https://github.com/kandref/Training-Fundamental-Pyhton.git
   ```
2. Install dependencies
   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the notebook with Jupyter
   ```bash
   jupyter notebook Segmentation_Credit_Card_Customer.ipynb
   ```
